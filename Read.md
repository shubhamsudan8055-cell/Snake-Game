
# 🐍 Snake Game in Python

A classic Snake Game built using **Python** and **Tkinter**. The player controls a snake that moves around the screen, collecting food to grow longer while avoiding collisions with the walls and its own body. This project demonstrates the fundamentals of GUI programming, object-oriented programming (OOP), event handling, and game development using Python.

## 🎮 Features

- Classic Snake gameplay
- Smooth keyboard controls
- Real-time score tracking
- Random food generation
- Collision detection
- Snake grows after eating food
- Game Over screen
- Clean and simple graphical interface
- Built with Object-Oriented Programming (OOP)

## 🛠 Technologies Used

- Python 3.x
- Tkinter (GUI Library)
- Random Module

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Run the Game

1. Clone the repository:

```bash
git clone https://github.com/your-username/Snake-Game.git
```

2. Navigate to the project folder:

```bash
cd Snake-Game
```

3. Run the program:

```bash
python snake_game.py
```

## 📸 Gameplay

The objective is simple:

- Eat the red food to increase your score.
- Every food eaten makes the snake longer.
- Avoid hitting the walls.
- Avoid colliding with your own body.
- Try to achieve the highest score possible!
